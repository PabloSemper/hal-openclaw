### Arquitectura Clave (Lo que usamos)
1. **Kwit** - Centro de control (OpenClaw Gateway)
2. **Multi-agente** - Múltiples agents en paralelo (HAL + Dev-Agent + Researcher)
3. **Memoria persistente** - Markdown files + facts.jsonl (así lo hicimos)
4. **Heartbeat** - Auto-check c/30min → nosotros c/3h (5 cron jobs)
5. **Dashboard web** - Interfaz de control (Mission Control)
6. **Multi-mensajería** - WhatsApp, Telegram, Discord, Slack, Signal, Teams

### Funciones Implementadas
✅ Gestión de archivos (read/write/organize)
✅ Ejecución de shell commands

✅ Email + Calendar (Gmail, Outlook, Google Calendar)
✅ Skills (extensible via marketplace)
✅ Heartbeat automático (proactivo)

### Riesgos de Seguridad (Mitigados en nuestra config)
⚠️ **Credenciales** → Variables de entorno (no en archivos)
⚠️ **Prompt injection** → Validar inputs de externos
⚠️ **Costos** → Limits de tokens preestablecidos
⚠️ **Acceso shell** → En NUC local, no máquina principal
⚠️ **Verificación de identidad** → Solo usuario autorizado
⚠️ **Skills maliciosos** → 26% tienen vulnerabilidades (Cisco report)

### Deployment (Nuestra config vs Video)

- Nosotros: NUC local (Anthropic/Claude) + SSH tunneling
